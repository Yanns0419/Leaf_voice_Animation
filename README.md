## 🎬 故事梗概

小女孩想要一颗新鲜的露珠，树叶小精灵从枝头看见，接下这份跨越世界的订单，翻山越岭送达。当露珠在日记本上绽放，两个世界完成了第一次温柔的对话。

#### 1.3 导出与导入流程
- **导出格式**：FBX（兼容Blender）
- **导入操作**：文件 → 导入 → FBX
- **结果**：6个模型成功导入Blender，进行后续材质、绑定、动画制作

### 2. 场景搭建
![场景布局](images/01_场景布局.png)

三个主要场景：树上、书房、星空，使用Collections集合管理。

### 3. 材质与着色
![材质节点](images/02_材质节点.png)

使用Principled BSDF + Mix Shader混合着色器，PBR物理材质。

### 4. 动画制作
![动画制作](images/03_动画制作.png)

骨骼绑定 + 关键帧动画，Dope Sheet管理多物体动画层。

### 5. 灯光设计
![灯光设置](images/06_灯光设置.png)

主光源为日光（6500K，强度8.55W），配合阴影焦散效果。

### 6. 渲染输出
![渲染设置](images/04_渲染设置.png)

Cycles引擎，GPU加速，视图采样1024，渲染采样500，开启降噪。

## 💡 总结

人与自然完成一次完美的请求与回应。和谐不是强制平等，而是各自做好自己的角色——人类表达需求，自然温柔送达，人类感恩铭记。露珠会蒸发，但笔记本和信封上的画证明：这份和谐，曾经真实存在。
